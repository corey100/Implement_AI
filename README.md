# Implement_AI


## Installation Requirements

1- Install PIP 
```apt-get install python3-pip```

2- Install Python 2
```apt-get install python2```

3- Install OpenCV with
```pip install opencv-python==2.4.9```
if this does not work install it manually from opencv.2.4.9
